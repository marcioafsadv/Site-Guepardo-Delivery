# 🐆 Guepardo Delivery - Ecossistema de Logística de Elite

Landing page moderna e de alta conversão para o ecossistema **Guepardo Delivery**. Acesse em: [guepardodelivery.com](https://guepardodelivery.com/)

Desenvolvida com foco em performance, estética premium e experiência do usuário (UX) intuitiva.

## 🚀 Tecnologias Utilizadas
- **HTML5 & Vanilla CSS3**: Arquitetura limpa para máxima velocidade.
- **Glassmorphism & Neon Design**: Estética futurista e premium.
- **Micro-interações**: Feedback visual fluido para o usuário.
- **WhatsApp Widget**: Canal de suporte direto integrado ("Guepardo de Plantão").

## 🛠️ Componentes do Ecossistema
- **Guepardo Central**: Painel administrativo para controle total da operação.
- **Savana do Guepardo**: App especializado para estabelecimentos e parceiros.
- **Guepardo Entregador**: App otimizado para a rotina do entregador.

## 📦 Como rodar o projeto
1. Clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. (Opcional) Utilize o VS Code Live Server para uma melhor experiência de desenvolvimento.
